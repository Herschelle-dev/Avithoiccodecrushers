Featured Apps

GitHub

Host and review code, manage projects, and collaborate with other developers.

LeetCode

Practice coding problems and prepare for technical interviews with timed challenges and contests.

Major League Hacking (MLH)

A student hackathon community that runs events, supports organizers, and hosts resources for hackers.

HackerRank

Practice coding, take part in contests, and complete skill assessments used by companies.

Codeforces

Competitive programming platform with regular contests and a large problem archive.

Stack Overflow

Community Q&A for programmers — find solutions or ask technical questions about code and tools.

Medium

Read and publish technical posts, tutorials, and essays about programming and product design.

LinkedIn

Professional networking — useful for internships, job hunting, and showcasing projects.
